#### 1. React.js는 어떤 기술이며 왜 활용되나요?

> 개발자에게 사랑받는 대표 이유로는 다음과 같습니다.
> 1. 컴포넌트 재사용성
> 2. 가상DOM을 이용하여 새로운 변경 사항에 대해서만 렌더링
